TL;DR - GDPR Summarizer

User Guide

Introduction

Welcome to TL;DR - GDPR Summarizer! This application is designed to help you summarize lengthy GDPR documents. You can either upload a PDF file or input text directly into the application, and it will provide a concise summary for you.

How to Use

Home Page

When you first open the application, you’ll be greeted with a simple interface. There are two main input methods: a text box for direct text input and a file upload button for PDF files.

Text Input

  1. Locate the text box labeled “Provide the input text (i.e. GDPR document contents) here…”
  2. Enter the text you want to summarize into this box.
  3. Click the “Summarize!” button.

File Upload

  1. Locate the file upload button.
  2. Click the button and select the PDF file you want to summarize.
  3. Click the “Summarize!” button.

Output

After clicking the “Summarize!” button, the application will process your input and display the summarized text in the output text box labeled “Summarized text will appear here”.

Tips

  • Make sure to provide either text or a PDF file before clicking the “Summarize!” button. If neither is provided, you will see an alert saying “No file or text provided”.
  • The application currently only supports PDF files for the file upload functionality.
